    The Bnai Zion Medical Center, established in 1922 as the first Jewish hospital in Haifa provides dedicated medical care,

  • Vision of the nursing management

  • 1. Oriented to a patient and his family: Courteous and accessible professional service available with compassion while maintaining the dignity and safety of the patient.

  • 2.Emphasis on human relations: Handling and acquiring inter-team communication skills and respect

  • 3.Professional Servicing: Providing treatment with the working procedures and learning from mistakes

  • 4.Satisfaction of the nursing team: Raising team satisfaction by encouraging fair learning and personal example

Answers

  • Question 1
1. Web1 focus on Connect Information- Companies controlled on the data in the web
Web2 focus on Connect People- people can search and perform their content to all the users
Web3 focus on Connect Knowledge- people can interact with others.
2. The data in Web1 was static, there is no connection between people
The data in Web2 it was more dynamic and friendly
The data in Web3 is like the data in Web2 but using AI
3. In Web1 the data was to read only- there was a list of all the data and a user can search specific site which exists in the data
In Web2 the user could search data in engine search
In Web3 the data processed before it appears to user and the search engine is more flexible




  • Question 2
Results of the query search "Side effect of medicines":
In web1, After the user search "side effect of medicines", The search engine displays only those sites named the search query expression
In Web2, After the user search "side effect of medicines", The search engine displays a list of sites that an exact query expression has appeared in
In Web3, After the user search "side effect of medicines", The search engine processes the information and also displays pages whose content is related to the query expression
